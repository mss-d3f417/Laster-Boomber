## Getting Started

### Prerequisites

This is an example of how to list things you need to use the software and how to install them.
* Install Required Libraries
  ```sh
  $ pip install user-agent
  $ pip install urllib3
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/mrd3f417/MR.Defalt-Language-Translator.git
   ```
2. Run 
   ```sh
   python Laster-Boomber.py
   ```
3. Enter Target Phone Number (ex:091200000)

4. Enter Time For Send Spam (default:0.1)

5. just Enter For Start Spam :) & Enjoy
